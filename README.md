# Kanmani

A rewrite of [Sunamu](https://github.com/NyaomiDEV/Sunamu). in python for windows. 

How to use. 
1. `git clone https://github.com/surajbhari/kanmani`
2. `pip install -r requirements.txt`
3. `python main.py`

Then Visit [localhost](http://localhost)
